💬 FAQ

常见问题

会被平台检测到并封号吗？

方案基于本地已登录浏览器运行，行为与真实用户完全一致。不是服务器端的批量 API 调用，平台无法区分。同时建议首先使用草稿模式验证内容。

能否同时管理多个账号？

为单机单账号设计。支持多账号轮换发布，每个账号独立 Cookie 隔离。

平台改版后还能用吗？

每个技能采用"多策略兜底"设计，每个 UI 元素有 3-5 个备选定位方案。享受改版后的快速适配服务。

需要技术人员来操作吗？

命令行操作需要基本技术能力。可提供可视化配置工具，运营人员即可一键发布。

支持哪些内容格式？

Markdown（.md）和纯文本（.txt）均支持。每个平台的适配器会自动处理格式转换。

封面图需要自己设计吗？

不需要。每个技能内置 Pillow 封面生成器，自动根据平台规格生成专业封面图。
